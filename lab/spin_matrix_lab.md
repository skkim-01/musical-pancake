# 2×2 스핀 행렬 대각화 실습

> "관측 연산자 → 대각화 → 고유값 = 관측 결과" 흐름을 직접 확인하는 실습

---

## 배경

### 전자의 스핀이란
전자는 자전하는 성질이 있는데, 이것을 스핀이라고 한다. 어떤 축 방향으로 측정하면 결과가 딱 두 가지만 나온다: **+1(업) 또는 -1(다운)**. 중간값은 없다.

### 측정 방향에 따라 연산자가 다르다
같은 전자의 스핀이라도 **어느 축으로 측정하느냐**에 따라 다른 연산자(행렬)를 사용한다. "어떤 연산자를 적용하느냐에 따라 다른 정보가 추출된다"는 것이 바로 이것이다.

### σ_z (z축 측정)

```
σ_z = [[1,  0],
       [0, -1]]
```

이것은 이미 대각행렬이다. 대각선에 1과 -1이 바로 보인다. 즉 대각화할 필요 없이 **고유값(관측 결과)이 눈에 보인다**: +1과 -1. 대각화란 복잡한 행렬을 대각행렬로 바꾸는 과정인데, 이미 대각행렬이면 할 일이 없으니 실습 의미가 없다.

### σ_x (x축 측정) ← 이걸로 실습한다

```
σ_x = [[0, 1],
       [1, 0]]
```

이것은 대각행렬이 아니다. 대각선이 0이고 다른 곳에 값이 있다. 그래서 **대각화를 직접 해봐야** 고유값(관측 결과)을 알 수 있다. 이것이 실습으로 적합한 이유이다.

대각화하면 고유값이 역시 +1과 -1로 나온다. 물리적으로 당연하다. 스핀을 z축으로 측정하든 x축으로 측정하든 결과는 업 아니면 다운이니까. **달라지는 것은 고유벡터(관측 후 상태)**이지, 고유값 자체는 같다.

---

## 실습 단계

### Step 1: 고유값과 고유벡터 구하기

```python
import numpy as np

# x축 스핀 연산자
sigma_x = np.array([[0, 1],
                     [1, 0]])

# 대각화: 고유값과 고유벡터 구하기
eigenvalues, eigenvectors = np.linalg.eigh(sigma_x)

print("고유값:", eigenvalues)
print("고유벡터:\n", eigenvectors)
```

**실행하기 전에 생각해볼 것:**
- 고유값이 몇 개 나올까? (힌트: 2×2 행렬이다)
- 고유값은 어떤 물리적 의미인가? (힌트: 관측 가능한 결과값)


### Step 2: 결과 해석하기

실행하면 고유값 두 개가 나온다. 이것이 **x축 방향으로 스핀을 측정했을 때 나올 수 있는 결과**다.

**확인해볼 것:**
- 고유값이 +1과 -1인가?
- 이것은 스핀 업(+1)과 스핀 다운(-1)에 해당한다
- 2×2 행렬이므로 관측 가능한 결과가 2개 — 행렬 차원과 관측 결과 수의 관계를 확인


### Step 3: A = PDP⁻¹ 직접 확인하기

```python
# P = 고유벡터 행렬 (이미 eigenvectors에 들어있음)
P = eigenvectors

# D = 고유값을 대각선에 놓은 행렬
D = np.diag(eigenvalues)

# P⁻¹
P_inv = np.linalg.inv(P)

# A = PDP⁻¹ 복원
reconstructed = P @ D @ P_inv

print("원래 행렬:\n", sigma_x)
print("PDP⁻¹로 복원:\n", np.round(reconstructed, 10))
```

**확인해볼 것:**
- 두 행렬이 같은가?
- 같다면 대각화가 성공한 것


### Step 4: 중첩 상태에서 관측 확률 계산하기

```python
# 전자가 z축 스핀 업 상태에 있다고 하자
spin_up_z = np.array([1, 0])

# 이 상태를 x축 스핀의 고유벡터로 분해
# 각 고유벡터에 대한 계수(내적) 구하기
coefficients = []
for i in range(len(eigenvalues)):
    evec = eigenvectors[:, i]
    coeff = np.dot(evec, spin_up_z)  # 내적!
    prob = abs(coeff)**2              # 확률 = |계수|²
    coefficients.append(coeff)
    print(f"고유값 {eigenvalues[i]:+.1f} 관측 확률: {prob:.4f}")
```

**확인해볼 것:**
- 확률의 합이 1인가? (노름 = 1 조건)
- 각 확률이 50%인가? (z축 업 상태에서 x축 측정하면 반반이 나와야 물리적으로 맞음)


### Step 5 (도전): y축 스핀 연산자

```python
# y축 스핀 연산자 — 복소수가 등장한다!
sigma_y = np.array([[0, -1j],
                     [1j,  0]])

eigenvalues_y, eigenvectors_y = np.linalg.eigh(sigma_y)

print("y축 스핀 고유값:", eigenvalues_y)
print("y축 스핀 고유벡터:\n", eigenvectors_y)
```

**확인해볼 것:**
- 고유값이 여전히 +1, -1인가?
- 고유벡터에 복소수(허수)가 포함되어 있는가?
- 복소수가 등장하는 이유: 양자역학에서 파동함수가 복소수 값을 가지기 때문

## 핵심 개념 체크리스트

실습을 마친 후 아래를 확인해보자:

- [ ] 고유값 = 관측 가능한 결과 (대각행렬 D의 대각 성분)
- [ ] 고유벡터 = 관측 후 상태 (행렬 P의 열)
- [ ] 행렬 차원 = 관측 결과의 수 (2×2 → 2개)
- [ ] 내적으로 확률 계산 (⟨고유벡터, 상태⟩의 절대값 제곱)
- [ ] 확률의 총합 = 1 (노름 조건)
- [ ] y축 연산자에서 복소수 등장 (켤레 대칭 공리가 필요한 이유)
